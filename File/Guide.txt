2- Install Required Packages

On Admin machine install SSH utilities.

sudo yum install openssh-clients -y

Check SSH:

ssh -V

5. Configure Passwordless SSH Login

Admin server should connect to CentOS servers without password.

Step 1 Generate SSH Key

On admin machine

ssh-keygen

Press Enter for default options.

Files created:

~/.ssh/id_rsa
~/.ssh/id_rsa.pub

Step 2 Copy Key to Servers

Example:

ssh-copy-id user@192.168.1.10

Repeat for all servers:(Create on all servers to login password less)

ssh-copy-id user@192.168.1.25
ssh-copy-id user@192.168.1.26
 
Step 3 Test SSH Login 

ssh user@192.168.1.26

If it logs in without password, configuration is successful.

6. Create Server List

Create file on admin machine:

vim servers.txt

Example content:

192.168.1.25
192.168.1.26
192.168.1.27

This file contains all servers that will receive patches.
